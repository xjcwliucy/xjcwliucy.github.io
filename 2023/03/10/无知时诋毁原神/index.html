题意:

原题面已经给得很清楚——题目传送门

显然，这是一道数论题。

做法：

对于 nnn 为偶数：输出 -1。

对于 nnn 为奇数：
构造 $F_{(2i\bmod n)}=i (i\in [0,n-1]\cap \mathbb{N}) $，两次遍历 CiC_iCi​ 并输出 FCiF_{C_i}FCi​​。

分析&做法解释：

  1. 首先不难发现，只要数据有解，无论怎样调换数组 ∗C^*C∗C 的排列，都可以通过改变 ∗A^*A∗A 与 ∗B^*B∗B 数组的排列而产生新的一组解，所以是否有解只与 nnn 的数值有关系。

  2. 通过观察样例，我们猜测：当 nnn 为奇数时，有解；当 nnn 为偶数时，无解（稍后给出证明）。

  3. 考虑如何构造排列 ∗A^*A∗A 和 ∗B^*B∗B。猜测，对于集合 F={i∈[0,n−1]∩N∣i≡2×k( mod n)}F=\lbrace i\in [0,n-1]\cap \mathbb N| i\equiv 2\times k (\bmod n)\rbraceF={i∈[0,n−1]∩N∣i≡2×k(modn)}，有 ∣F∣=n|F|=n∣F∣=n，

    即 ∀i,j∈[0,n−1]∩N\forall i,j \in [0,n-1]\cap \mathbb N∀i,j∈[0,n−1]∩N，若 i≠ji\neq ji​=j，则有 (2×i) mod n≠(2×j) mod n(2\times i) \bmod n \neq (2\times j) \bmod n(2×i)modn​=(2×j)modn。

    若如此，可以使对于 ∀i∈[0,n−1]∩N\forall i\in [0,n-1]\cap \mathbb N∀i∈[0,n−1]∩N，满足 Ai=BiA_i=B_iAi​=Bi​，构造无疑会方便许多。（这个猜想当然是对的，稍后给出证明）

对猜想的证明（没错现在就是稍后）：

首先证明：

∀i,j∈[0,n−1]∩N\forall i,j \in [0,n-1]\cap \mathbb N∀i,j∈[0,n−1]∩N，若 i≠ji\neq ji​=j，有 (2×i) mod n≠(2×j) mod n(2\times i) \bmod n \neq (2\times j) \bmod n(2×i)modn​=(2×j)modn。

证明如下：

首先假设存在 $i,j\in[0,n-1]\cap\mathbb N $，满足 i≠ji\neq ji​=j，且 (2×i) mod n=(2×j) mod n=r(2\times i) \bmod n = (2\times j) \bmod n = r(2×i)modn=(2×j)modn=r。不妨设 i<ji<ji<j。

设:

(2×i)=k1×n+r(2 \times i)=k_1 \times n + r(2×i)=k1​×n+r；

(2×j)=k2×n+r(2 \times j)=k_2 \times n + r(2×j)=k2​×n+r；

易证：

0<k2−k1<20<k_2-k_1 < 20<k2​−k1​<2，

即： k2−k1=1k_2-k_1=1k2​−k1​=1。

即： (2×j)−(2×i)=n(2 \times j)-(2 \times i)=n(2×j)−(2×i)=n。

因为 nnn 为奇数，所以矛盾，证明完毕。

同时，若 nnn 为偶数，此不矛盾，且可以举出实例，故无解。
